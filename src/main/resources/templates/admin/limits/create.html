Информация
  • Минимальный рост: указывается в сантиметрах (например: 140 см)
  • Максимальный рост: должен быть больше минимального (например: 180 см)
  • Минимальный возраст: указывается в полных годах (например: 12 лет)
  • Только мужчины: ограничивает доступность мотоциклов с этим лимитом для женщин
  • После создания ограничение нужно будет привязать к мотоциклам
  • Ограничения автоматически фильтруют доступные мотоциклы при бронировании
